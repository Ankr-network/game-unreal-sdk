MintItems function is used to mint a batch of items to the user and requires the user confirmation through wallet such as metamask.

The function requires parameters described below and returns nothing. Inside the function, A POST request is sent to the Mirage API. The request needs a json body containing a device_id, contract_address, abi_hash, method and args. The format is described in the body section below. string data will be received in json response for a "ticket".

Parameters
abi_hashThe hash of the abi string of the contract.
ResultA callback delegate that will be triggered once a response is received with data.
Attention
YOUR_ITEM_ADDRESS(s) and YOUR_ITEM_QUANTITY(s) represent each other respectively.
