Detailed Description

AdvertisementManager provides functionalities to load and show ads for various platforms.

Member Function Documentation

◆ StartSession()

void UAdvertisementManager::StartSession ( )

StartSession function is used to start a new advertisement session.

The function does not require parameter and returns nothing. The function is called once at the start of the app and once session gets expired.

Parameters
ResultA callback delegate that will be triggered once a response is received with a data.

FString  _deviceId,
FString  _publicAddress,
FString  _language 
)

InitializeAdvertisement function is used to set the app id, device id, public address and language for the ads.

The function requires parameters and returns nothing.

Parameters
_appIdThe app id for the advertisement.
_deviceIdThe device id that the AnkrSDK is using.
_publicAddressThe wallet address of the user logged in.
_languageThe language to be used for the advertisement.
Attention
This function also loads native ads libraries for iOS and Android.
